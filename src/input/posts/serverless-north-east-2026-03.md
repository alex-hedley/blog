### Lee Gilmore

AWS Practise Lead @ Leighton & AWS Hero

**How to build an Order Management System for a global brand on AWS in 14 weeks**

Go behind the scenes of a high-speed, 14-week journey building an OMS for a global brand on AWS using event-driven principles. Learn how we navigated the "Orchestration vs. Choreography" debate, modeled complex order data, and tackled the hard lessons of brand-specific routing. Whether you're curious about secure webhook authentication or scaling notification patterns, this session offers a roadmap for delivering enterprise-grade commerce logic at pace.

### Matheus Guimaraes

Senior Developer Advocate @ AWS

**Agentic Microservices: The Next Evolution of Microservices Architecture**

Microservices architecture is evolving. AI agents are reshaping how backend systems are built and consumed. Discover Agentic Microservices, a new evolution of microservices architecture, along with emerging patterns such as Microservices as Tools (MAT) and Agentic Monoliths. We also explore why serverless is a natural fit for agentic architectures and watch an agentic microservice in action.
